# GPP X content rotation drafts — golf page first

Brand voice:
- act like a golf content/meme page first, product page second
- 80% broad golf content: observations, history, stats, course weirdness, player narratives, mildly edgy takes
- 15% subtle pool-picks angle: mention picks/standings/pool sweat without a hard sell
- 5% hard CTA: create pool / guide link / pricing
- hard rule: no golf-word salad. Every post needs one actual joke, insight, complaint, fact, or recognizable golf-culture situation
- hard rule: if the post still works after swapping in random golf nouns, delete it
- hard rule: do not mention picks unless the post is explicitly in the 15% pool-picks bucket or 5% CTA bucket
- useful enough that a golf sicko respects it
- a little opinionated/edgy, never mean or spammy
- no group-chat framing
- no emojis

Target mix: 80% golf content, 15% subtle pool-picks, 5% hard CTA.
Approval workflow: drafts only. Pick winners, then we can wire posting commands later.
